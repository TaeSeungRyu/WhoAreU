# WhoAreU

> Windows에서 실행 중인 프로그램이 **누구인지** 알려주는 트레이 앱.

작업 관리자만으로는 "이 프로세스가 뭐 하는 녀석이지?"가 잘 보이지 않습니다.
WhoAreU는 지금 내 PC에서 돌고 있는 사용자 앱들의 **출처, 설치일, 디스크 용량, 메모리 사용량**을 한 화면에 모아 보여줍니다.

---

## 주요 기능

- 현재 실행 중인 사용자 앱 프로세스 목록 (Windows 시스템 프로세스 자동 제외)
- 각 프로세스의 게시자(Publisher), 실행 파일 경로, 설치일 표시
- 디스크 설치 용량과 실시간 메모리 사용량을 함께 표시
- 검색 / 정렬 가능한 대시보드
- **트레이 상주** — 평소에는 트레이 아이콘으로만 보이고, 필요할 때 열어서 확인
- **포터블 실행** — 설치 과정 없이 exe 파일 하나로 실행

## 사용법

### 트레이 아이콘 메뉴

| 메뉴 | 동작 |
|------|------|
| 확인하기 | 대시보드 창을 엽니다 |
| 종료 | 앱을 완전히 종료합니다 |

대시보드 창의 X 버튼은 트레이로 숨기기일 뿐, 종료가 아닙니다. 완전히 끄려면 트레이 메뉴의 **종료**를 사용하세요.

### 대시보드

- 상단 검색창에서 프로세스/게시자 이름으로 필터링
- 컬럼 헤더 클릭으로 정렬
- 새로고침은 수동(버튼)과 자동(주기 갱신) 두 가지 모드 지원 — 토글로 전환

## 실행

```powershell
# 의존성 설치
yarn install

# 개발 모드 실행
yarn start

# 포터블 exe 빌드
yarn build
# → dist/WhoAreU-<version>-portable.exe
```

## 지원 환경

- Windows 10 / 11 (x64)
- 관리자 권한 없이 동작하지만, 일부 시스템 프로세스 정보는 권한이 부족하면 표시되지 않을 수 있습니다.

## 기술 스택

Electron + 바닐라 HTML/CSS/JS. 시스템 정보는 PowerShell을 통해 수집하며, 네이티브 모듈 의존성은 두지 않았습니다.
